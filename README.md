# CISC4631_FinalProject
My final project for CISC4631

This project focuses on detecting fraudulent credit card transactions using data mining models. It compares Logistic Regression and Random Forest on an imbalanced dataset.

The dataset used in this project is the Credit Card Fraud Detection dataset from Kaggle. Due to file size limitations, it is not included in this repository.

It can be found at: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

## Requirements:
Python 3.x
pandas
scikit-learn
matplotlib

## Instructions:
1. Download the dataset from the link above.
2. Place the file `creditcard.csv` in the same folder as the notebook.
3. Open the notebook and run all cells in order.
